# 💡 Segundo projeto - Consumo de Energia
Este projeto foi criado com a intenção de calcular o consumo de energia de um utensílio elétrico. A linguagem utilizada foi Python.

O modelo de cálculo é simples, porém funcional: a potência do aparelho é multiplicada pelas horas de uso diário, depois por 30 dias e, ao final, o resultado é dividido por 1000 para obter o consumo em kWh.

## ▶️ Como executar
1. Instale o Python em seu computador
2. Baixe ou clone este repositório
3. Execute o arquivo `.py` no terminal ou no VS Code

## 🧠 Fórmula utilizada
Consumo = (potência × horas × 30) / 1000
